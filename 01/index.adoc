= Help

== Stateful

* Start

[source,bash]
----
$ node .src/stateful.js
----

* Connect using telnet

[source,bash]
----
$ telnet 127.0.0.1 8124
----

* Controls

** Open new order
+
[source,bash]
----
open
----

** Add item to opened order
+
[source,bash]
----
add mango
----

** Process opened order
+
[source,bash]
----
process
----

== Stateless

* Start

[source,bash]
----
$ node .src/stateless.js
----

* Connect using telnet

[source,bash]
----
$ telnet 127.0.0.1 8124
----

* Controls

** Open new order
+
[source,bash]
----
open
----

** Add item to opened order of id 0
+
[source,bash]
----
add 0 mango
----

** Process opened order of id 0
+
[source,bash]
----
process 0
----
